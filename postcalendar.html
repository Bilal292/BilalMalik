Life in Weeks Calendar

This is a static website hosted using cloudflare.

The website allows users to input their date of birth and other information that they want to include in their life calendar.

Using css media print I made the calendar printable in a good format for users to print out.

Javascript includes some error handling so that the program doesn't break if the users provides wrong input.

Technologies:
  • - Javascript
  • - HTML
  • - CSS
  • - Github
  • - CloudFlare
